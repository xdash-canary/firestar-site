# Voltarian Technologies

Premium website and Discord bot hosting solutions with lightning-fast performance and 99.9% uptime guarantee.

## Features

- **Website Hosting**: High-performance web hosting with SSD storage, free SSL certificates, and one-click WordPress installation
- **Discord Bot Hosting**: Reliable bot hosting with 24/7 uptime, DDoS protection, and easy Git integration
- **99.9% Uptime Guarantee**: Industry-leading uptime with SLA backing
- **24/7 Expert Support**: Round-the-clock technical support
- **Lightning Fast Performance**: NVMe SSD storage and optimized servers
- **Advanced Security**: Enterprise-grade DDoS protection and security measures

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser

## Build & Deploy

Build the application for production:
```bash
npm run build
```

Start the production server:
```bash
npm start
```

## Technology Stack

- **Next.js 14**: React framework with App Router
- **TypeScript**: Type-safe development
- **Tailwind CSS**: Utility-first CSS framework
- **Framer Motion**: Smooth animations and interactions
- **Lucide React**: Beautiful icon library
